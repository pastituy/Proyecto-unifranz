# Logs Directory

This directory contains all output files generated by the Baneco QR Generator.

## Contents

### Log Files
- **`qr.log`** - Main application log file containing all API operations, requests, and responses

### QR Results (JSON)
- **`qr_result_*.json`** - Complete API responses for each QR generation
  - Contains QR ID, QR image (base64), response code, and messages
  - Named with transaction ID for easy tracking

### QR Images (PNG)
- **`qr_image_*.png`** - Decoded QR code images ready to use
  - PNG format
  - Named with transaction ID matching the JSON result

## File Naming Convention

All files use the transaction ID in their names:
```
qr_result_TXN-20251212204437.json
qr_image_TXN-20251212204437.png
```

Where `TXN-20251212204437` is the transaction ID in format: `TXN-YYYYMMDDHHmmss`

## Automatic Cleanup

Consider implementing automatic cleanup for old files:

```python
# Example: Delete files older than 30 days
import os
import time
from pathlib import Path

logs_dir = Path('docs/logs')
current_time = time.time()
days_to_keep = 30

for file in logs_dir.glob('qr_*'):
    if file.is_file():
        file_age = current_time - file.stat().st_mtime
        if file_age > (days_to_keep * 86400):  # 86400 seconds in a day
            file.unlink()
```

## Security Note

This directory may contain sensitive information:
- QR codes with payment information
- Transaction IDs
- Account details (encrypted in logs)

Ensure proper access controls are in place for this directory in production environments.
